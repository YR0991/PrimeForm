# PrimeForm — Marketing site

De marketing-/landingssite voor PrimeForm staat in deze map. Alles wat bij de site hoort (Next.js App Router, sectie-componenten, content) zit hier.

- **Build & run:** `npm install` → `npm run dev` (of `npm run build` + `npm start`)
- **Deploy:** Bouw vanuit deze map (bijv. Vercel: root directory = `site`)

Commit en push je wijzigingen zoals gewoonlijk; de hele repo (inclusief `site/`) wordt gecommit en gepusht. De site is deze sectie.

## Structuur

- `app/` — Next.js App Router (routes, pages, layout)
- `components/` — React-secties (Hero, Steps, KPI, …)
- `content/` — Copy (brand, sitemap, copy_blocks, pages, evidence)
